# PlanDelta AI v0.2 Execution Pack

This pack is an additive execution contract for the existing public `abdullahahsen05/plandelta-ai`
repository and stable `v0.1.0` release.

## Files

- `AGENTS.md` — autonomous execution, product, safety, Git, AWS, and final repair rules.
- `START_AGENTIC_V0_2.md` — Codex entry point and resume protocol.
- `PLAN_AGENTIC_V0_2.md` — product/architecture scope and success criteria.
- `PHASES_AGENTIC_V0_2.md` — sequential Phase 12–22 implementation tracker.
- `docs/ARCHITECTURE_V0_2.md` — runtime topology, graph, state, and tool bounds.
- `docs/CREDENTIALS_V0_2.md` — safe reuse of credentials and new configuration.
- `docs/DATABASE_RAG_V0_2.md` — pgvector/RAG/conversation/run/citation schema.
- `docs/API_CHAT_V0_2.md` — public/internal endpoints, SSE, errors, and quotas.
- `docs/GUARDRAILS_V0_2.md` — prompt injection, citations, limits, and failures.
- `docs/EVALS_V0_2.md` — dataset, metrics, thresholds, and live/fake modes.
- `docs/DOMAIN_PROFILES_V0_2.md` — construction and engineering-schematic scope.
- `docs/CHAT_UX_V0_2.md` — workspace, citations, RFI, sample, accessibility.
- `docs/OBSERVABILITY_COST_V0_2.md` — redacted traces, metrics, and AWS budget.
- `docs/FINAL_VERIFICATION_V0_2.md` — comprehensive test-and-repair loop.
- `docs/APPLICATION_EVIDENCE_V0_2.md` — source-backed recruiter answer template.
- `CODEX_PROMPT.md` — short prompt to provide with the pack.

## Installation

Extract/copy the pack into the repository root. It intentionally replaces the root `AGENTS.md` with
the v0.2 instructions while preserving the completed original `PLAN.md`, `PHASES.md`, and
documentation as v0.1.0 history. Commit the execution contract in its own small commit on
`feat/agentic-v0.2`.

Do not extract secrets, credentials, model caches, or generated output into the repository.
